# Laravel React User Roles

This project is a full-stack application using Laravel (backend) and React (frontend) for user role management.

## Structure
- `backend/` - Laravel API
- `frontend/` - React app

## Setup

### 1. Clone the repository
```
git clone <repo-url>
```

### 2. Backend Setup
```
cd backend
composer install
cp .env.example .env
php artisan key:generate
```

### 3. Frontend Setup
```
cd frontend
npm install
npm run dev
```
